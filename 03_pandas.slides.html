$$ Pandas $$¶

Pandas es un paquete open source de python para representar estructuras de datos. Permite llevar a cabo el anlisis de datos de manera fcil e intuitiva similar a como se hace en R. Es muy flexible para la integracin con otras librerias aumentando la productividad. "Python con pandas est siendo usado en gran variedad de mbitos acadmicos y comerciales, incluyendo Finanzas, Neurociencia, Economa, Estadstica, Publicidad, Web Analytics y mucho ms".
